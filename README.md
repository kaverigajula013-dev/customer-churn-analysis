# Customer Churn Analysis

## Project Overview
This project analyzes customer behavior to identify churn patterns and factors influencing customer retention.

## Tools Used
Python
Pandas
Seaborn
Matplotlib

## Objectives
- Analyze customer churn distribution
- Identify patterns in customer behavior
- Visualize churn trends using data visualization

## Key Insights
- Customers with higher monthly charges show higher churn rates
- Visualization helps identify churn patterns
